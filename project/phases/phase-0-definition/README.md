# Phase 0 — Project Definition

- **Status:** 🟢 Complete *(reviewed & approved 2026-06-15)*

## Goals

Define *who* the project is (name, vision, positioning, scope) and *how* it will be built
(documentation-first, AP-driven, phase-based, with visible trackers). No code; no architecture or
protocol design (that is Phase 2).
